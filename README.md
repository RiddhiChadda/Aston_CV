# AstonCV – Dynamic CV Database Website

## Overview
AstonCV is a dynamic web application built using PHP and MySQL that allows users to register, log in, create and manage their CVs, and explore other users' profiles.

The system demonstrates server-side development concepts including authentication, database interaction, session management, and secure form handling.

---

## Features

### Public / Core Features
- User registration system
- Secure login and logout
- View all CVs (protected)
- View full CV details
- Search CVs by:
  - Name
  - Key programming language

### Registered User Features
- Create an account
- Log in securely
- Edit and update personal CV
- Complete profile after registration

---

## CV Structure

Each CV includes:

- Name
- Email
- Phone
- Location
- Key Programming Language
- Skills
- Profile
- Education
- Experience
- Projects
- URL Links
